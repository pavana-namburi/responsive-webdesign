Introduction

JavaScript is the language of the web, allowing interactive and dynamic content on websites. It runs inside the browser and can update HTML, CSS, and handle user interactions.

Variables

Variables store data values. Use let, const, or var.

let name = "Pavana"; // changeable
const pi = 3.14;     // fixed value
var age = 20;        // old style (avoid)

Data Types

JavaScript has different data types: string, number, boolean, object, array, undefined, null, and symbol.

let text = "Hello";      // String
let num = 100;           // Number
let isOnline = true;     // Boolean
let fruits = ["apple","mango"]; // Array
let student = {name:"Pavana", age:20}; // Object

Operators

Operators perform operations on variables and values.

let a = 10, b = 5;
console.log(a + b); // 15 (Addition)
console.log(a - b); // 5  (Subtraction)
console.log(a * b); // 50 (Multiplication)
console.log(a / b); // 2  (Division)

Functions

Functions are reusable blocks of code.

function greet(user) {
  return "Hello " + user;
}
console.log(greet("Pavana"));

Objects

Objects group related data and functions.

let student = {
  name: "Pavana",
  age: 20,
  greet: function() { return "Hi!"; }
};
console.log(student.greet());

Arrays

Arrays store multiple values in a single variable.

let colors = ["red", "green", "blue"];
console.log(colors[0]); // red
colors.push("yellow");  // add new item

Loops

Loops help execute code multiple times.

for (let i = 0; i < 5; i++) {
  console.log("Number: " + i);
}

let i = 0;
while (i < 3) {
  console.log("Hi " + i);
  i++;
}

Conditions

Conditions let you control the flow of code.

let score = 85;
if(score >= 90){
  console.log("Grade A");
} else if(score >= 75){
  console.log("Grade B");
} else {
  console.log("Grade C");
}

DOM

Document Object Model (DOM) lets JS interact with HTML elements.

document.getElementById("demo").innerText = "Updated!";
document.querySelector("h1").style.color = "blue";

Events

Events allow code to run when users interact with the page.

<button onclick="sayHello()">Click Me</button>

function sayHello(){
  alert("Hello World!");
}

ES6 Features

Modern JavaScript (ES6) introduced many features like arrow functions, template literals, and classes.

// Arrow function
const add = (x, y) => x + y;

// Template literal
let user = "Pavana";
console.log(`Welcome ${user}!`);

// Class
class Car {
  constructor(name) {
    this.name = name;
  }
  drive() {
    return this.name + " is driving";
  }
}
let car1 = new Car("Tesla");
console.log(car1.drive());
