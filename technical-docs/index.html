Introduction

JavaScript is a versatile programming language used for web development.

Variables

Variables store data values. Use let, const, or var.

Functions

Functions are blocks of code that can be reused.

function greet(name) {
  return "Hello, " + name;
}
Loops

Loops let you repeat actions. Example: for and while loops.

Objects

Objects store key-value pairs and are fundamental in JS.
