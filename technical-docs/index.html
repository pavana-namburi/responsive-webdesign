Introduction

JavaScript (JS) is a high-level, dynamic programming language used primarily for web development. It allows developers to create interactive and dynamic web pages. JS can run in browsers, on servers using Node.js, and even in mobile and desktop applications.

Key features of JavaScript include:

  • Event-driven programming
  • Dynamic typing
  • Prototype-based object orientation
  • First-class functions

Syntax

JavaScript syntax is the set of rules that define a correctly structured JS program. Statements are generally terminated with a semicolon (;), and blocks of code are defined using curly braces {}.

Basic syntax elements include variables, operators, functions, and expressions.


// Example
let greeting = "Hello World!";
console.log(greeting);

Variables

Variables are used to store data values. JavaScript provides three ways to declare variables:

  • let: Block-scoped variable, can be updated.
  • const: Block-scoped constant, cannot be updated.
  • var: Function-scoped variable (older, avoid in modern code).

// Declaring variables
let name = "Pavana";
const PI = 3.1416;
var age = 21;
console.log(name, PI, age);

Functions

Functions are reusable blocks of code that can take inputs (parameters) and return an output. They help make code modular and maintainable.

Functions can be declared in multiple ways:

  • Function declaration
  • Function expression
  • Arrow function

// Function declaration
function greet(user) {
    return "Hello " + user + "!";
}
console.log(greet("Pavana"));

Loops

Loops allow executing a block of code multiple times until a condition is met. Common types of loops include:

  • for loop: Runs a specific number of times
  • while loop: Runs while a condition is true
  • do...while loop: Executes at least once, then checks condition

// For loop example
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

Events

Events are actions that occur in the browser, such as a user clicking a button or pressing a key. JavaScript can respond to these events using event listeners.


// Event listener example
document.getElementById("myBtn").addEventListener("click", () => {
    alert("Button clicked!");
});
